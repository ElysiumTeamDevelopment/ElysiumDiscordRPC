# Creator-Defined Statements (CDS) для Discord RPC

ElysiumDiscordRPC поддерживает удобный синтаксис команд (CDS) для управления статусом Discord прямо из скриптов Ren'Py, без необходимости использовать Python-блоки или вызовы функций с `$`.

## Установка

Для использования CDS убедитесь, что файл `01-discord-rpc.rpy` находится в папке `game/libs/` вашего проекта

## Использование

### Основные команды

#### 1. Произвольный статус
Устанавливает произвольный текст для состояния и деталей.

```renpy
# Синтаксис: discord custom "State" "Details"
discord custom "Исследует мир" "Глава 2"
```

#### 2. Статус диалога
Устанавливает статус чтения диалога с указанием персонажа и сцены.

```renpy
# Синтаксис: discord dialogue "Character Name" "Scene Name"
discord dialogue "Эйлин" "Комната"
```

#### 3. Игровой процесс
Устанавливает статус активной игры.

```renpy
# Синтаксис: discord in_game "Chapter Name" "Character Name"
discord in_game "Глава 1" "Эйлин"
```

#### 4. Пауза
Устанавливает статус "На паузе".

```renpy
# Синтаксис: discord paused
discord paused
```

#### 5. Загрузка
Устанавливает статус "Загрузка...".

```renpy
# Синтаксис: discord loading
discord loading
```

#### 6. Главное меню
Устанавливает статус главного меню.

```renpy
# Синтаксис: discord main_menu
discord main_menu
```

#### 7. Меню
Устанавливает статус нахождения в меню (например, настройки, галерея).

```renpy
# Синтаксис: discord menu "Menu Name"
discord menu "Настройки"
```

## Примеры использования в скрипте

```renpy
label start:
    # Начало игры
    discord custom "Начало игры" "Пролог"

    "Добро пожаловать в игру!"

    # Общение с персонажем
    discord dialogue "Эйлин" "Гостиная"
    e "Привет! Как дела?"

    # Переход к игровому процессу
    discord in_game "Глава 1" "Поиск улик"
    
    # ... игровой процесс ...

    # Пауза (например, перед сложным выбором)
    discord paused
    menu:
        "Идти налево":
            pass
        "Идти направо":
            pass

    return
```

## Преимущества CDS

1.  **Чистота кода**: Команды выглядят как нативные инструкции Ren'Py.
2.  **Удобство**: Нет необходимости писать `$` и скобки `()`.
3.  **Читаемость**: Сразу понятно, что делает команда.
